PIR Motion Sensor Camera App

By Malick, Vinicius, Samuel, and Hichem

Project Description

This project utilizes a PIR (Passive Infrared) motion sensor and a Raspberry Pi camera to detect human presence. Whenever the PIR sensor detects motion, the Raspberry Pi camera captures a photo of the scene. The captured photos are stored in an SQLite database, and you can view all the pictures on one page. The system also generates a graph displaying the time and date of each picture. Clicking on a specific picture allows you to view it in a separate page.
